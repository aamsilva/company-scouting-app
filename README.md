# Company Scouting App

## 📋 Sobre o Projeto

Aplicação web desenvolvida para a **Autoridade Tributária (AT)** - Portugal, para auxiliar técnicos de inspeção tributária no terreno.

### 🎯 Objetivo

Facilitar a identificação e localização de empresas durante inspeções tributárias em campo, através de uma interface intuitiva integrada com Google Maps e Google Location API v2.

### ✨ Funcionalidades Principais

- 🗺️ **Mapa Interativo** - Visualização de empresas no mapa com Google Maps
- 📍 **Geolocalização** - Localização precisa de empresas por endereço
- 🔍 **Pesquisa Inteligente** - Busca por NIF, nome ou endereço
- 📱 **Interface Mobile-Friendly** - Optimizado para uso em tablets/smartphones
- 🔄 **Integração NIF** - Consulta automática de dados empresariais

### 🏗️ Tecnologias

- **Frontend:** React 19 + Vite
- **Maps:** Google Maps JavaScript API + Google Location API v2
- **Dados:** Integração com NIF.pt
- **Deploy:** GitHub Pages
